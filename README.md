# Projet Site Web École - Rihab el Marjane

Bonjour Madame,

Voici le dépôt GitHub contenant l'intégralité du code source et l'architecture de notre application web.

Le projet est structuré de la manière suivante :
📁 admin : Contient tous les fichiers liés à l'administration du site.
📁 teacher : Espace réservé aux enseignants.
📁 config : Fichiers de configuration et connexion à la base de données.
 📁 assests /Images : Stockage des médias, logos et fichiers audio.
 📄 index.php / login.php / logout.php : Fichiers principaux séparés à la racine pour l'accès au site.
